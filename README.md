## Run Locally

Clone the project

```bash
  git clone https://github.com/Elishaya13/argentBank_p13_OC_CDA
```

Go to the project directory

```bash
  cd argentBank
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
